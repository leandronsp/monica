# monica

Playing with Assembly x86_64.

## Requirements

* x86_64
* GNU/Linux
* NASM 2.16.01
* ld 2.38
* GNU debugger (gdb) 12.1 
* strace 5.16

## Running

```bash
$ ./build.sh hello
$ ./target/hello

# or simply

$ ./run.sh hello
0
```

## Why this name

"Monica" is a bad try on pun with the word "mnemonics".
